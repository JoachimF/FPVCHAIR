OpenTX lite NextStepRC, pour 9X, 9XR, Gruvin9X et MEGA2560 DIY

Fichiers .cmd :
- Initialisation_M2560 : initialise la carte MEGA2560 (bootloader +
  fusibles + eeprom) via un programmateur USBASP
- MakeFW : compile le firmware OpenTX
- OpenTX_Companion : lance Companion
- OpenTX_Simulateur : lance le simulateur de firmware

Pour changer les paramètres du fichier MakeFW.cmd en fonction de votre
configuration matérielle (type de carte, LCD, options, etc., voir listing
thématique dans fichier "FW_options.pdf") :
- clic droit sur le fichier
- clic sur "Modifier"
- enregistrer après modification

Certaines configurations de paramètres de compilation du firmware ne fonctionnent pas,
générant une erreur de flashage. Sachant que celles proposée en base dans ce dossier est
"full options" et fonctionne à tous les coups, il est recommandé de ne pas la changer,
la radiocommande fonctionnant très bien en l'absence des modules externes associés, comme
la carte SD. Pour les fonctions, type GVAR ou autres courbes, libre à chacun de les utiliser,
ou pas, suivant son besoin.
Sinon, une configuration "lite" fonctionnant à coup sûr : SDCARD=NO RTCLOCK=NO VOICE=NO HELI=NO
